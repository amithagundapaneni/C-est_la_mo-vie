# C'est La Mo-Vie

> *Movies chosen for who you are — and how you feel tonight.*

This is a mood-aware movie recommendation system that understands **both your long-term taste and your current viewing intent** using natural-language embeddings and hybrid semantic search.

---

## ✦ What Makes It Different

Most recommenders ask: *"What did you watch last?"*  
We ask: *"What do you feel like tonight?"*

```
"I loved La La Land and Past Lives.
 Tonight I want something hopeful and romantic but not emotionally devastating."
```

The system understands both the **taste profile** (derived from favourite films) and the **current intent** (the free-text mood description) and blends them into a single semantic recommendation.

## ⚡ Quick Start

### 1. Install dependencies
```bash
cd afterglow
pip install -r requirements.txt
```

### 2. (Optional) Download TMDB data from Kaggle
Place any of these in `data/raw/`:
- [`tmdb_5000_movies.csv`](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
- `tmdb_5000_credits.csv` (same dataset)
- [`movies_metadata.csv`](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)

> **No download needed to run the demo** — a 100-movie sample is included.

### 3. Run the setup pipeline
```bash
python setup_pipeline.py
```
This runs ETL → embedding → clustering. Takes ~2–5 min on first run (model download included). Subsequent runs use cache.

### 4. Launch the app
```bash
streamlit run app/streamlit_app.py
```

## 🧠 Technical Notes

### Embedding model
`all-MiniLM-L6-v2` from `sentence-transformers` — fast (384-dim), strong semantic understanding, runs entirely locally.

### Hybrid blending
```
score = 0.6 × cosine(intent_embedding, movie_embedding)
      + 0.4 × cosine(taste_embedding,  movie_embedding)
```
Taste embedding = mean of embeddings of favourite movies (normalised).

### Vibe clusters
KMeans (k=10) on movie embeddings. Labels are assigned heuristically; the system uses them to enrich explanations and could be extended for cluster-filtered browsing.

### Explanation engine
Pure rule-based: no external LLM API. Uses genre text, vibe cluster, rating, and favourite title to generate a human-readable sentence.
